# Yoga Delight – AI-Powered Yoga Personalization

Yoga Delight is an intelligent wellness platform that combines AI + computer vision to deliver personalized yoga recommendations, real-time pose correction, and a fully adaptive practice experience.

## Features

- **AI-generated personalized yoga sequences**
- **Webcam-based pose detection**
- **Real-time posture correction**
- **Yoga recommendations**
- **Clean, responsive UI**
- **Beautiful motion-based visuals**

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **TensorFlow.js**
- **Tailwind CSS**
- **Custom Fonts & UI Components**

## Getting Started

### Prerequisites

Make sure you have Node.js installed on your system.

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/yoga-delight.git
cd yoga-delight
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open your browser and navigate to:

```
http://localhost:3000
```

## Key Features Breakdown

### 1. Personalized Recommendations

The AI analyzes user preferences, fitness levels, and goals to generate customized yoga sequences.

### 2. Real-Time Pose Detection

Using your webcam and MoveNet's pose estimation model, the app tracks your body movements in real-time.

### 3. Instant Feedback

Get visual cues and corrections as you practice, helping you perfect your form and prevent injuries.

### 4. Responsive Design

Fully optimized for desktop, tablet, and mobile devices with a beautiful, modern interface.
